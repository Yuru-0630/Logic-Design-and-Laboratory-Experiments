## 功能説明：

  玩家通過左右移動來躲避掉落物，分爲有限模式和無限模式。
  
  有限模式：30秒内被球碰到的次數低於3次就會顯示成功畫面，碰到三次就會馬上結束顯示失敗畫面。
  
  無限模式：一直進行游戲，一直到被碰到三次為止。
